# QualityCat Academy

Platforma szkoleniowa z celowymi bugami do nauki testowania.

## Struktura

- `apps/api` — backend (API)
- `apps/web` — frontend
- `packages/shared` — wspoldzielone typy i DTO
- `docs/` — dokumentacja
- `infra/` — konfiguracja Docker

## Uruchomienie

```bash
pnpm install
pnpm dev
```
